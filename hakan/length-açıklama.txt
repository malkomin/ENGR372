This page acts a unit converter for length units such as meter(m),kilometer(km),inches(in) and so on. The user inputs a number to the left box labeled as "from" and chooses what unit it wants to convert from and to using the select drop down menus. Afterwards the user clicks on calculate and gets the desired result. Also if the user also wants they can submit it to the main calculator to do operations with. In addition to that there is also a table of the unit conversions if the user wants to calculate by hand.
